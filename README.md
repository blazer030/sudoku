# Sudoku

A progressive web app (PWA) Sudoku game built with Vue 3 and TypeScript.

## Features

- Three difficulty levels (Easy, Medium, Hard)
- Guaranteed unique-solution puzzles
- Note mode and auto-notes
- Undo, erase, and conflict highlighting
- Game save/restore with timer
- Statistics tracking (best times, win rate, day streak)
- Offline support via PWA

## Tech Stack

- **Framework**: Vue 3 + TypeScript
- **Build**: Vite + vite-plugin-pwa
- **Styling**: Tailwind CSS v4
- **State**: Pinia
- **Testing**: Vitest + @vue/test-utils
- **Icons**: Lucide Vue Next

## Development

```bash
npm install
npm run start     # Dev server
npm test          # Run tests (watch mode)
npm run lint      # ESLint
npm run build     # Production build
npm run preview   # Preview production build
```
